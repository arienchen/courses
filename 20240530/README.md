## 課程主題：
1. 數位轉型｜數位稽核
2. 稽核｜資料

* 數位轉型、數位稽核及稽核數位轉型

* 數據分析 Data Analysis 及工具介紹
* Excel 稽核技巧篇
* 資料清理方法及原則
* 資料處理幫手：Pentaho Data-Integration 工具
* 稽核所需的資料處理技能
